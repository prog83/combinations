# Combinations

Combinations

## Installation

Download or git clone [source code](https://gitlab.com/prog83/combinations).

Use the package manager [npm](https://www.npmjs.com/get-npm) to install.

```bash
npm i
```

## Usage

### Development

```bash
npm run start
```

### Testing

```bash
npm run test
```

### Production

```bash
npm run build
```
